## Overview
### Description
A game that asks users to guess a number between 1-100. Users have 10 chances to guess correctly. Will you guess the correct number?! 

### App Evaluation
- **Category:** Entertainment / Games
- **Desktop:** This app is primarily developed for desktop, easy to use and fun to play on.
- **Story:** Users can guess a number and try to get a best high score!
- **Market:** Any individual could choose to use this app, from a small child to an elderly person.
- **Habit:** This app could be used as often as the user wants. It depends on how much they want to play, and what they want to do.
- **Scope:** The game is simple, yet addicting as it keeps you guessing! 

## Product Spec
### 1. User Stories 

**Required Must-have Stories**

* User enters a number
* User is prompted if the number is too small or too big
* If user guesses correctly, they win. Their high score,  and score go up.
* If user guesses incorrectly, they have 9 tries left. One life is subtracted each time until they reach 0 and lose. 

**Optional Nice-to-have Stories**

* More information about each user
* Overall high score saved in a database
* Database with user's information
* Badges/levels to reward users

### 2. Screen Archetypes

* Number guess
 - Users input a number to guess
* Winning 
 - Users are presented with a celebratory sound and green screen as their score goes up.
* Losing
 - Users are presented with a red screen as they guess for the 10th time 


### 3. Navigation

**Tab Navigation** 

* Home

**Flow Navigation** (Screen to Screen)
* Guess number -> read feedback -> guess number => repeat -> win/lose -> play again
